Secure Web Server Infrastructure on AWS

Project Overview

This project demonstrates building a secure web server infrastructure using Amazon VPC, Public Subnet, Security Groups, and Amazon EC2 with Apache HTTP Server.

---

AWS Services Used

- Amazon VPC
- Public Subnet
- Internet Gateway
- Route Table
- Amazon EC2
- Security Groups
- Apache HTTP Server

---

Implementation Steps

1. Created custom VPC
2. Configured Public Subnet
3. Attached Internet Gateway
4. Configured Route Table
5. Created Security Groups
6. Launched EC2 Instance
7. Installed Apache Web Server
8. Hosted static website

---

Learning Outcomes

- AWS Networking
- VPC Architecture
- EC2 Management
- Security Group Configuration
- Apache Web Server Setup
